getAll( [isPublic] [, refresh] [, user])

Get all workshops that the current session's user has permissions for (or all publicly listed workshps). The function assembles a list of workshop ids form the users permissions to query Salesforce. The queried fields from Salesforce are as follows:

[
 "Id",
 "Name",
 "Start_Date__c",
 "End_Date__c",
 "Course_Manager__c",
 "Billing_Contact__c",
 "Event_City__c",
 "Event_Country__c",
 "Organizing_Affiliate__c",
 "Public__c",
 "Registration_Website__c",
 "Status__c",
 "Host_Site__c",
 "Workshop_Type__c",
 "Language__c"
]

The query is ordered by 'Start_Date__c'.

Parameters:
Name Type Argument Default Description
isPublic boolean <optional>
 false

Get Only public workshops (skips permission check)
refresh boolean <optional>
 false

Force the refresh of the cache
user any <optional>

The user to filter permissions for (isPublic === false); user needs permissions[] and roles[].permissions[]
